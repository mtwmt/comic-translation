    🎨 AI 漫畫翻譯工具 - 使用說明
═══════════════════════════════════════════════════

📌 這是什麼？
───────────────────────────────────────────────────
一個使用 Google Gemini AI 自動將日文漫畫翻譯成繁體中文的工具。
只需要簡單幾個步驟，就能完成漫畫的翻譯！


🚀 使用步驟
───────────────────────────────────────────────────

【第一步】取得 Gemini API Key
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 用瀏覽器開啟：https://makersuite.google.com/app/apikey
2. 使用 Google 帳號登入
3. 點擊「Create API Key」按鈕
4. 複製產生的 API Key（長得像：AIzaSyA...）

⚠️ 重要：API Key 是您的私密資訊，請勿分享給他人！


【第二步】執行程式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
雙擊「漫畫翻譯工具.exe」開啟程式


【第三步】設定
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
在開啟的視窗中：

1. 【API Key】
   - 將剛才複製的 API Key 貼到「Gemini API Key」欄位
   - 如果忘記怎麼取得，可點擊「❓ 如何取得」按鈕

2. 【輸入資料夾】
   - 點擊「瀏覽...」按鈕
   - 選擇存放日文漫畫圖片的資料夾

3. 【輸出資料夾】
   - 點擊「瀏覽...」按鈕
   - 選擇要儲存翻譯結果的資料夾


【第四步】開始翻譯（必看！）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. 點擊「🚀 開始翻譯」按鈕
2. 程式會自動處理所有圖片
3. 處理進度會顯示在進度條
4. 完成後可點擊「📁 開啟輸出資料夾」查看結果


🎯 進階功能（選用）
───────────────────────────────────────────────────

【人名對照表】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
如果您希望特定的人名統一翻譯，可以在此設定。

格式：原文=中文（一行一個）

範例：
サトシ=小智
ピカチュウ=皮卡丘
John=約翰
メアリー=瑪麗

說明：
- 程式會確保這些人名永遠翻譯成您指定的中文
- 適合用在有固定角色的漫畫系列
- 以 # 開頭的行會被忽略（註解用）


【額外提示詞】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
可以給 AI 額外的指示，讓翻譯更符合您的需求。

範例：
- 保持可愛活潑的語氣
- 使用台灣的用語習慣
- 擬聲詞保留日文原文

說明：
- 這些指示會加到基本的翻譯指令後面
- 不會影響基本的翻譯功能
- 以 # 開頭的行會被忽略（註解用）


💡 使用技巧
───────────────────────────────────────────────────
✓ 建議使用解析度較高的圖片，翻譯效果會更好
✓ 每張圖片處理時間約 10-30 秒，視複雜度而定
✓ 已翻譯過的圖片會自動跳過，可重複執行
✓ 可以隨時點擊「⏸ 停止」中斷處理
✓ 程式會自動記住上次選擇的資料夾


📋 支援的圖片格式
───────────────────────────────────────────────────
✓ PNG (.png)
✓ JPEG (.jpg, .jpeg)
✓ WebP (.webp)


⚠️ 注意事項
───────────────────────────────────────────────────
1. API 使用可能會產生費用
   - Gemini API 有免費額度，但超過後會收費
   - 請注意您的使用量

2. 網路連線
   - 需要穩定的網路連線才能使用
   - 處理時請確保網路暢通

3. 翻譯品質
   - AI 翻譯可能不完美，建議人工檢查
   - 複雜的圖片可能需要多次嘗試

4. 隱私安全
   - API Key 不會被儲存或上傳
   - 每次使用都需要重新輸入（安全考量）
   - 圖片會傳送到 Google 伺服器進行處理


❓ 常見問題
───────────────────────────────────────────────────

Q: 執行時出現錯誤怎麼辦？
A: 請檢查：
   - API Key 是否正確
   - 網路連線是否正常
   - 輸入資料夾中是否有圖片檔案

Q: 翻譯結果不理想怎麼辦？
A: 可以嘗試：
   - 使用更高解析度的圖片
   - 在「額外提示詞」中給予更明確的指示
   - 調整人名對照表

Q: 處理速度很慢？
A: 這是正常的，AI 處理需要時間
   每張圖片約 10-30 秒
   請耐心等候

Q: 可以同時處理多個資料夾嗎？
A: 目前不支援，請一個一個資料夾處理
